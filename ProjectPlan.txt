## 🧹 WEEK 1 — Python + MySQL

### Step 1: Explore & Clean (cleaning.py)

```python
import pandas as pd
import numpy as np
from sqlalchemy import create_engine

# ── Load ────────────────────────────────────────────────────
df = pd.read_csv('data/raw/transactions.csv')

print(df.shape)
print(df.dtypes)
print(df.isnull().sum())
print(df.duplicated().sum())
```

```python
# ── Clean ───────────────────────────────────────────────────

# Drop duplicates
df.drop_duplicates(inplace=True)

# Fix date column to proper datetime
df['TransactionDate'] = pd.to_datetime(df['TransactionDate'])

# Extract useful time features
df['Month'] = df['TransactionDate'].dt.month
df['DayOfWeek'] = df['TransactionDate'].dt.day_name()
df['Hour'] = df['TransactionDate'].dt.hour

# Standardize text columns
df['TransactionType'] = df['TransactionType'].str.strip().str.title()
df['MerchantCategory'] = df['MerchantCategory'].str.strip().str.title()
df['Location'] = df['Location'].str.strip().str.title()

# Handle missing values
df['MerchantCategory'].fillna('Unknown', inplace=True)

print("Cleaned shape:", df.shape)
```

### Step 2: Feature Engineering

```python
# ── Flag Suspicious Transactions ────────────────────────────
# This is where your cybersecurity background becomes an asset

# Flag 1: High-value transactions (top 5% of amounts)
threshold = df['TransactionAmount'].quantile(0.95)
df['IsHighValue'] = (df['TransactionAmount'] > threshold).astype(int)

# Flag 2: Late night transactions (11PM - 4AM) — higher fraud risk
df['IsLateNight'] = df['Hour'].apply(
    lambda x: 1 if (x >= 23 or x <= 4) else 0
)

# Flag 3: Balance drops significantly after transaction
df['BalanceDrop'] = df['AccountBalance'].shift(1) - df['AccountBalance']
df['IsSuspicious'] = (
    (df['IsHighValue'] == 1) & (df['IsLateNight'] == 1)
).astype(int)

# Customer spending category
df['SpendingTier'] = pd.cut(
    df['TransactionAmount'],
    bins=[0, 500, 2000, 10000, float('inf')],
    labels=['Low', 'Medium', 'High', 'Very High']
)
```

### Step 3: Save Cleaned File

df.to_csv('data/cleaned/transactions_clean.csv', index=False)

### Step 5: Write SQL Queries (queries.sql)

**A) Customer Spending Behavior**
```sql
-- Total spend and avg transaction per customer
SELECT 
    CustomerID,
    COUNT(*) AS total_transactions,
    ROUND(SUM(TransactionAmount), 2) AS total_spend,
    ROUND(AVG(TransactionAmount), 2) AS avg_transaction,
    MAX(TransactionAmount) AS max_transaction
FROM transactions
GROUP BY CustomerID
ORDER BY total_spend DESC
LIMIT 20;
```

**B) Spending by Category**
```sql
-- Which merchant categories drive the most volume?
SELECT 
    MerchantCategory,
    COUNT(*) AS transaction_count,
    ROUND(SUM(TransactionAmount), 2) AS total_spend,
    ROUND(AVG(TransactionAmount), 2) AS avg_spend
FROM transactions
GROUP BY MerchantCategory
ORDER BY total_spend DESC;
```

**C) Monthly Trend**
```sql
-- Month-wise transaction volume and value
SELECT 
    Month,
    COUNT(*) AS total_transactions,
    ROUND(SUM(TransactionAmount), 2) AS monthly_volume
FROM transactions
GROUP BY Month
ORDER BY Month;
```

**D) Suspicious Transaction Summary**
```sql
-- How many suspicious transactions per location?
SELECT 
    Location,
    COUNT(*) AS suspicious_count,
    ROUND(SUM(TransactionAmount), 2) AS suspicious_volume
FROM transactions
WHERE IsSuspicious = 1
GROUP BY Location
ORDER BY suspicious_count DESC;
```

**E) High-Value Customers**
```sql
-- Customers with highest average transaction amounts
SELECT 
    CustomerID,
    Age,
    ROUND(AVG(TransactionAmount), 2) AS avg_spend,
    COUNT(*) AS total_txns
FROM transactions
GROUP BY CustomerID, Age
HAVING total_txns > 5
ORDER BY avg_spend DESC
LIMIT 10;
```

**F) Transaction Type Breakdown**
```sql
-- Credit vs Debit split by spending tier
SELECT 
    TransactionType,
    SpendingTier,
    COUNT(*) AS count,
    ROUND(SUM(TransactionAmount), 2) AS total_amount
FROM transactions
GROUP BY TransactionType, SpendingTier
ORDER BY TransactionType, total_amount DESC;
```
